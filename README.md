Project Name: Task Analytics Dashboard 

Description:
A web-based task management app enhanced with data-focused features. Users can add, edit, and delete tasks while tracking additional details like due dates, categories, and estimated completion times. The app collects and stores data in a MySQL database and provides analytics dashboards showing task completion rates, category breakdowns, and time-based statistics. Built with Node.js, Express, and vanilla JavaScript for frontend interactions, the project demonstrates full-stack development, SQL query mastery, and basic data visualization skills.

Key Features:

CRUD operations for tasks with extended metadata (due dates, categories, estimated time)

Inline editing and dynamic task rendering

Analytics dashboards with SQL-powered statistics and charts

Optional CSV/JSON import to practice ETL-style data ingestion

Focus on real-world SQL queries, grouping, and reporting

Tech Stack: Node.js, Express, MySQL, vanilla JavaScript, HTML/CSS, optional Chart.js for visualizations

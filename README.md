# Task Analytics Dashboard

## Description
A web-based task management app enhanced with data-focused features. Users can add, edit, and delete tasks while tracking additional details like due dates, categories, and estimated completion times. The app collects and stores data in a MySQL database and provides analytics dashboards showing task completion rates, category breakdowns, and time-based statistics. Built with Node.js, Express, and JavaScript for frontend interactions, and Python for visualizations, the project demonstrates full-stack development, SQL, and basic data visualization skills.

## Key Features
- CRUD operations for tasks with extended metadata (due dates, categories, estimated time)
- Inline editing and dynamic task rendering
- Analytics dashboards with Python-powered statistics and charts

## Tech Stack
Node.js, Express, MySQL, JavaScript, HTML/CSS, Python for visualizations

## To set up the database
1. Open MySQL or your SQL client.  
2. Run the commands in `schema.sql` to create the database and table.  
3. Update your `.env` file with the correct database credentials.
